[프로젝트 배경 및 목표]
- 본 프로젝트는 특정 장소(유적지, 전시장, 이벤트 공간 등)를 방문한 관람객에게 특별한 경험을 제공하는 것을 목표로 합니다. 관람객이 자신의 스마트폰으로 QR코드를 스캔하면, 별도의 애플리케이션 설치 과정 없이 웹 브라우저를 통해 즉시 증강현실(AR) 콘텐츠를 체험할 수 있도록 구현합니다. 제공된 캐릭터를 활용하여 관람객이 현실 공간을 배경으로 캐릭터와 함께 사진을 촬영하고, 이를 손쉽게 저장 및 공유하게 함으로써 바이럴 마케팅 효과와 방문객 만족도 제고를 기대합니다.

[과업 범위]
1. 수행 범위
- QR코드 스캔 → 웹사이트 연결 → 카메라 화면 활용 → 캐릭터 등장 → 포토 촬영 및 저장
- 별도 앱 설치 없이 브라우저(WebAR) 기반으로 구동

2. 상세 기능 요구 사항
1. 체험 시작
- QR코드 스캔 → 웹페이지 접속 시 카메라 접근 권한 요청 → 촬영 화면 표시
2. 캐릭터 등장
- 배경 : 실제 카메라 화면
- 캐릭터 PNG : 점점 커지며 등장 + 간단한 텍스트 소개(자막)
3. 사진 촬영 기능
- 관람객 + 캐릭터 PNG + 배경 합성 후 촬영 버튼으로 사진 저장 가능
- 촬영된 사진 상단에 관련 정보(유적지/장소 안내) 자동 표출
4. 저장 및 공유
- 촬영 결과물을 스마트폰에 저장
- SNS 공유 버튼(선택 사항)

[지원 디바이스 및 디자인]
1. 지원 디바이스
- PC Web (Chrome, Edge 등 최신 버전 브라우저 호환성 확보)
- Mobile Web (iOS, Android 최신 버전의 기본 브라우저에 최적화된 반응형 웹)
